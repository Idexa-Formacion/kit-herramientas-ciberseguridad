## [0.1.0] - 2026-04-28

### Añadido

- README inicial con presentación del toolkit y catálogo de cursos de Idexa Formación.
- Scripts de red: `quick-scan.sh`, `port-audit.sh`, `ssl-check.sh`.
- Scripts de auditoría: `audit-linux.sh`, `password-policy-check.sh`, `find-suid.sh`.
- Scripts de hardening: `harden-ssh.sh`, `disable-services.sh`, `firewall-baseline.sh`.
- Scripts de respuesta a incidentes: `triage-linux.sh`, `collect-logs.sh`, `ioc-search.sh`.
- Checklists de ISO 27001:2022, hardening de Linux y respuesta a incidentes.
- Plantillas de Política de Seguridad, Informe de Incidente y Plan de Respuesta.
- Workflow de CI con ShellCheck, escaneo de secretos y markdown lint.
- Documentación de instalación, contribución y política de seguridad.
